# Yorushika-Musical-Fingerprint-Analysis
アーティストの音楽的特徴の数値化と理論的再現に関する研究

ヨルシカの楽曲群を対象に、その音楽的特徴（手癖）を統計的・数値的に解析し、得られた理論に基づいた楽曲再現の可能性を追求する。

## 実験手法
### 1. 収集
ヨルシカの楽曲をMIDIデータに変換
- Demucsでフル音源を分離
- basic-pitchで各パートをMIDI化
- midoでノイズや極端な強弱を
- Pythonでノートをグリッドに揃える

### 2. 分析
music21を中心にアーティストの手癖を数値化
- music21
  - コード進行を度数表記に変換、遷移確率を算出
  - メロディの移動確率を算出
  - シンコペーションの頻度やフレーズの開始・終了位置を調べる
  - コードの構成音とメロディの関係を調べる
- ライブラリ外
  - BPM、キー、拍子、音域の集計
  - 音符密度の算出
    
### 3. 生成
- キー・拍子・BPM・音域・音符密度などを統計を元にランダムに決定
- コード遷移確率に基づき8小節のコード進行を生成
- これらを条件としてMagentaで8小節のサビを生成する
- MIDIをmidoで書き出す
- CubaseにMIDIを読み込み調整
